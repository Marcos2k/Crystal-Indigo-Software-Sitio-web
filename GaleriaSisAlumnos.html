                Crystal indigo Software

Sistema de admnistración de alumnos versión 1.1


Descripción:

El sistema se encarga de administrar alumnos realizando altas, bajas y modificaciones, registrando asistencias, administrando profesores, clases, genera pagos imprimiendo comprobantes y genera reportes incluso.


Caracteristicas:

Fácil de usar: Cuenta con una interfaz grafica rica y amigable por el cual facilita su uso.
Inteligente: Permite llevar el control de las clases con los datos de los alumnos y profesores.
Liviano: Solo pesa 10 MB! Agil, liviano y totalmente eficaz con buena performance.
Flexible: El software es totalmente modificable a las necesidades del negocio, sin que los cambios afecten a su funcionamiento implementando nuevos modulos a pedido del usuario cuando este lo desee.


Requisitos tecnicos:

Como minimo el equipo deberá contar con 1 GB de RAM y un procesador superior a 1 GHz. Recomendado: 2 GB RAM y un procesador 1.90 GHz. Sistema Operativo: Windows 7 Home Premium Service Pack 1, Windows 7 Professional Service Pack 1, Windows 7 Ultimate Service Pack 1, Windows 8.1 o superior.


Precio:

$400 ARS por mes - Licencia de uso.
